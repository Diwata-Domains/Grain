## Goal

Write the full first draft of the research brief, following the outline above
and grounding every claim in the gathered sources.

## Do this

1. Write each section in the order given by the outline.
2. For every factual claim, cite the supporting source inline (reuse the source
   labels from the source list). Where a source is still a placeholder, keep the
   `[source: ...]` / `[needs source]` marker rather than inventing a citation.
3. Keep the prose tight and decision-oriented; lead with the bottom line.
4. Do not introduce claims that the outline and sources do not support.

Write only from the outline and sources above. Do not assume network access.

## Write the artifact

Write the complete draft to `04-draft.md` as finished Markdown prose with inline
source references.
