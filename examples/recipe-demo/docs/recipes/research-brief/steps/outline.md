## Goal

Turn the framing and the gathered sources into a section-by-section outline for
the brief. This step produces structure, not finished prose.

## Do this

1. Define the brief's sections in order (e.g. summary, background, findings,
   analysis, recommendation).
2. Under each section, list the bullet points it will cover.
3. Map each major bullet to the supporting source(s) from the source list above.
4. Ensure every framed sub-question is answered by at least one section.
5. Note any section that is currently unsupported so the draft step can flag it.

Use only the framing and sources above. Do not introduce new claims here.

## Write the artifact

Write the outline to `03-outline.md` as a nested Markdown list of sections and
bullets, each annotated with its supporting source.
