## Goal

Compile the sources and evidence that answer the framed question and its
sub-questions. This step produces a source list, not prose.

## Do this

1. For each sub-question in the framing above, identify the evidence needed.
2. Record each source as an entry with: a short label, what it establishes,
   and (if known) where it comes from. Use placeholders like
   `[source: <description>]` where a citation must be filled in later.
3. Note the relative strength of each source (strong / moderate / weak) and any
   conflicts between sources.
4. Flag remaining gaps where no source yet exists.

Do not assume live network access. Capture what you know and mark unverified
claims explicitly so a later step can resolve them.

## Write the artifact

Write the annotated source list to `02-sources.md` as a Markdown list grouped
by sub-question, with strength and gap notes.
